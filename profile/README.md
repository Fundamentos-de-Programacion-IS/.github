# Fundamentos de Programación  
> Curso 2025 – Universidad de Sevilla  

Bienvenidos al repositorio de presentaciones del curso **Fundamentos de Programación**. En este espacio encontrarás el conjunto completo de diapositivas utilizadas durante las sesiones, organizadas por tema y módulo.

También será la plataforma donde se publicarán los entregables y ejercicios realizados durante las prácticas y teoría. 
---

## Contenido del curso  
A continuación, un resumen de las sesiones disponibles:

| Módulo | Tema | Contenido |
|--------|------|----------|
| 00     | Presentación del curso | Objetivos, metodología y evaluación. |
| 01     | Introducción | Conceptos básicos y primer contacto. |
| 02     | Expresiones y tipos | Tipos de datos, operadores y evaluación. |
| 03     | Condicionales y bucles | Estructuras de control de flujo. |
| 04     | Funciones | Definición, argumentos y retorno. |
| 05     | Secuencias: tuplas y listas | Colecciones ordenadas y operaciones. |
| 06     | Entrada y salida | Captura de datos, escritura en archivos. |
| 07     | Buenas prácticas en Python | Estilo PEP 8, documentación y herramientas de soporte. |
| 08     | Objetos y dataclasses | Modelado de entidades y uso de `dataclass`. |

> Todas las presentaciones están disponibles en la [página del curso](https://fundamentos-de-programacion-is.github.io/slides/) para su consulta e impresión.

---

## Contribuciones y cambios

* Las actualizaciones de las diapositivas (correcciones menores, mejoras, nuevas sesiones) se registran mediante *commits* que incluyen las fechas de modificación.
* Si deseas sugerir mejoras o colaborar (por ejemplo, correcciones de tipografía, añadidos de ejercicios, traducciones), abre un *issue* o envía un *pull request*.
* Por favor, sigue la estructura existente de carpetas y nomenclatura para mantener la uniformidad.
